Quick Start Guide

Follow these steps to get up and running with GitHub Projects:

1. Create Your GitHub Account

  1. Go to github.com and click Sign up
  2. Use your IJHS-registered email, choose a username, and set a password
  3. Verify your email by clicking the link sent to your inbox
  4. (Optional) Enable Two-Factor Authentication under Settings ‚Üí Security

2. Accept Our Repository Invitation

  1. Find the invite from ijhs-spark-2025@github.com in your email
  2. Click View invitation and then Accept invitation

3. Access Your Project Board

  1. Log in to GitHub and open the ijhs-spark-2025 repository
  2. Select the Projects tab and choose your project (Greenfluence / PhishBlaster / AIdea)

Frequently Asked Questions

How do I create and label a task?
Click New issue, enter a clear title, pick your project label, and optionally select a milestone. Then click Submit new issue.
What do the board columns mean?
  • Backlog: Unprioritized tasks
  • Ready: Committed for this sprint
  • In Progress: Actively working
  • In Review: Awaiting feedback
  • Done: Completed tasks
How do I move a task forward?
Drag the issue card to the next column (Ready ‚Üí In Progress ‚Üí In Review ‚Üí Done) as you progress.
Where can I find more detailed instructions?
Check out our CONTRIBUTING.md and WORKFLOW.md files for full guidelines and workflow details.
What if I need to change a task's details?
You can edit the task details directly in the issue card. Click on the card, then select Edit to update the title, description, labels, or milestones.
How do I assign tasks to team members?
When creating or editing a task, you can assign it to team members by selecting their GitHub username in the Assignees section. This helps clarify who is responsible for each task.
Can I add comments or feedback on tasks?
Yes! You can add comments directly on the issue card. This is a great way to provide feedback, ask questions, or discuss details with your team. Just click on the task and use the comment box at the bottom to add your thoughts.
How do I filter or search for specific tasks?
Use the search bar above the board to filter by keywords, labels, assignees, or milestones. For example, type label:greenfluence is:open to see all open Greenfluence tasks.

How do I close a task when it's done?
You can either drag the card to the Done column, or open the issue and click Close issue. Either action will mark it complete.
What if I need to reprioritize tasks mid-sprint?
  • In Table view, drag the tasks up or down to adjust priority.
  • Or edit the Due date or a custom Priority field so it sorts above others automatically.
